Repositorio: Un repositorio es un sitio centralizado que funciona como
una carpeta en la nube donde se almacena y gestiona todo el código, 
los archivos y el historial de cambios de un proyecto de software.

Commit: es un punto en el historial de tu proyecto que guarda tu progreso
por si en algún momento necesitas volver a un punto en especifico lo 
puedas hacer fácilmente. cada vez que generamos un commit git almacena datos como: autos, fecha, hora, mensaje, identificado único y el código.

Push: Es un comando que se traduce como empujar en este caso se trata de
una acción de "subir" o enviar tus archivos y cambios en ellos desde tu 
computadora hasta tu repositorio en git hub. Esta acción sincroniza tu código desde tu computadora, hasta el sitio en la web donde tus compañeros pueden descargar avances de tus proyectos.
